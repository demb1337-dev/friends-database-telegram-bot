# friends-database-telegram-bot
This Telegram bot allows you to store and manage friend information in a SQLite database. You can add, delete, edit, and search for friends by name, as well as view a full list of all your friends. This project is designed for a portfolio and demonstrates skills in Python, Telebot, and SQLite.

How to run
1. Install dependencies:
pip install pyTelegramBotAPI 
2. Copy the bot.py and test.db files (if you have a database) to one directory.
3. Insert your Telegram bot token into the code.
4. Start the bot:
python bot.py (or python3 if you are using linux)
5. Open Telegram and type /start.
